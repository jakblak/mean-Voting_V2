## Voting App ##

# Running the App
`npm install`

open up mongodb using *mongod*

`gulp copy:libs`   -   to copy angular into lib folder

`npm start`  

- runs nodemon on port 3000. 

## Features ##
- add posts with title, link and description
- upvote & downvote
- add comments + expand comments
- log in / sign up
